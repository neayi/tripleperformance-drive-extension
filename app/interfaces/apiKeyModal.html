Obtenir une clé API

Vous avez besoin d'une clé API de Google pour accéder à Gemini, notamment pour la gestion des quotas. La clé est gratuite, mais vous pouvez payer pour obtenir des limites plus élevées si nécessaire.
Inscrivez-vous ici et cliquez sur "Obtenir une clé".

Consentement au partage avec Google Gemini™

En enregistrant une clé API et en envoyant des requêtes aux modèles d’IA Gemini™ de Google, vous partagerez des données avec Google. Vous ne partagerez que les informations de la cellule dans laquelle vous exécutez une requête Gemini. Ces données sont soumises aux Conditions d’utilisation de Gemini™ de Google.
